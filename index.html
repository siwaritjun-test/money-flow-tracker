Methodology. True fund-flow data is proprietary, so flows are estimated: each asset's dividend/split-adjusted return over the period is compared with its group average; the excess return is multiplied by average daily dollar volume (capped at 2.5× group median to prevent one giant asset dominating). All assets in a group are aligned on common trading dates, so every return covers the same calendar window (Bitcoin trades weekends; ETFs don't). Negative scores are treated as money leaving, positive as money entering, and outflows are distributed to inflows pro-rata to build the flow map. The regime bar scores classic risk gauges (risk-appetite ratios over ~1 month, VIX level, 10y−3m yield-curve slope) as risk-on / risk-off votes; copper and gold use front-month futures, yields use CBOE indices. Data: Yahoo Finance daily bars. This is an analytical visualization, not investment advice.
